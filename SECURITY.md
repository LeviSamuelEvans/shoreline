# Security Policy

Shoreline is early-stage software, but it handles SoundCloud OAuth credentials and access tokens, so security reports are welcome.

Please do not open public issues for suspected vulnerabilities. Instead, report security issues privately by emailing the maintainer at the contact address listed on the GitHub profile for this project.

When reporting an issue, include:

- A concise description of the problem.
- Steps to reproduce, if available.
- The affected version or commit.
- Any relevant logs with credentials, tokens, stream URLs, and account identifiers removed.

The maintainer will acknowledge valid reports as soon as practical and coordinate fixes before public disclosure.
